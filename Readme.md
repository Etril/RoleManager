Ce projet a pour objectif de créer un backend en C# avec le framework ASP.NET Core, pour gérer plusieurs users (admin, manager et user) avec 
des permissions d'accès et d'edition différentes. En fond, l'objectif est d'utiliser une "Clean Architecture" en Domain/Infrastructure/Application/API, 
et de mettre en place des tests voir une pipeline CI/CD avec Github Actions. 

Technologies et outils utilisés: 
-Xunit pour les tests.
-Moq pour les données.
-JWT Auth. 
-FluentAssertions pour les assertions. 
-Coverlet pour le coverage. 
-GitHub Actions pour le CI/CD
